# BeautiQ Frontend

React web application for the BeautiQ beauty services platform.

## Tech Stack
- React 18
- Material-UI
- Socket.IO Client
- Stripe Elements

## Deployment
- Docker containerization
- Nginx for production serving
- GitHub Actions CI/CD
- AWS hosting

## Setup

```bash
npm install
npm start
```

## Docker

```bash
docker-compose up --build
```

## Features
- Customer booking & payments
- Service provider dashboard
- Admin management panel
- Real-time chat
- Notifications
- Feedback with sentiment analysis
